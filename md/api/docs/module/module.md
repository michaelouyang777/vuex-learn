* Module类
  
  描述：

  - constructor构造函数

    描述：

    初始化属性。包括runtime、_children、_rewModule、state，这4个属性


  - get 

    描述：

    获取namespaced


  - addChild

    描述：

    添加子module


  - removeChild

    描述：

    移除某个子module


  - getChild

    描述：

    获取某个子module


  - hasChild

    描述：

    是否存在某个子module


  - update

    描述：

    更新module。包括namespaced、actions、mutations、getters
    

  - forEachChild

    描述：
    
    遍历子属性


  - forEachGetter

    描述：

    遍历getter


  - forEachAction

    描述：

    遍历action


  - forEachMutation

    描述：

    遍历mutation
