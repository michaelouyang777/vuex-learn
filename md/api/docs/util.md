1. find方法

   描述：

   查找数组内的数据，并返回第一条。

  
2. deepCopy方法

   描述：

   深拷贝对象。


3. forEachValue方法

   描述：

   遍历对象。


4. isObject方法

   描述：

   判断是否对象。


5. isPromise方法

   描述：

   判断是否promise。


6. assert方法

   描述：

   判断是否存在对象，没有则抛出error。


7. partial方法

   描述：

   传入一个函数和参数，再用一个函数将它们封装起来，返回一个新的而无需传参的函数。使用的是柯里化函数缓存了参数。
